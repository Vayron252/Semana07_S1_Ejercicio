Responsive Design

1. Explicar las vertientes o enfoques mas usados para el RWD

Mobile First

Es un concepto de diseño web enfocado para móviles, donde inicialmente se crea una página web para dispositivos pequeños, para adaptarla luego a las necesidades de un navegador de escritorio

Desktop First

Es un concepto de diseño web enfocado para navegadores de escritorio, donde inicialmente se crea una página web para desktop, para luego adaptarla a las necesidades de un dispositivo más pequeño.

2. Escribir un ejemplo para cada vertiente o enfoque

Mobile First

Ejemplo:
Se realiza un menú de navegación para el móvil, este empieza verticalmente, pero a medida que se va alargando horizontalmente, este va acomodandose para escritorio, hasta que llega ocupar el 90% de la pantalla.

Desktop First

Ejemplo:
Se crea una galeria de imagenes que cuando ocupa toda la pantalla ocupa 4 x 3, a medida que se va acortando la pantalla, esta se va acomodando para que ocupen 2 x 8, y hasta que en un dispositivo movil ocupe 1 x 12
